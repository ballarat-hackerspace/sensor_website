Frequently Asked Questions

What's my core ID?

This is a 24-character ID made up of letters and numbers used to identify your sensor.


I've forgotten my core ID. How can I find out what it is?

You can send us a message using the contact page. Please include both your full personal name and your organisation's name.


I've removed a sensor but would like to restore it. Is this possible?

The only way to restore a sensor is to re-register it. Just use the same location information you did the first time and it will be back to normal.
